# iCloud Duplicate Cleaner

A simple Python script to help find and clean up duplicate files in an iCloud folder or other local directory.

## What it does

This script scans a folder, looks for duplicate files, and helps you identify or remove them so your storage stays tidier.

It is useful if:

- iCloud Drive has ended up with repeated copies of files
- you have downloaded the same files more than once
- you want a cleaner folder structure
- you want a quick Python tool instead of doing it by hand

## Features

- Scans a chosen folder for duplicates
- Compares files by content or file properties
- Shows duplicate groups clearly
- Can help you decide what to keep and what to remove
- Saves time compared with checking folders manually

## Requirements

- Python 3.x
